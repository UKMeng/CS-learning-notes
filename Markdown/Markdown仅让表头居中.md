# Markdown仅让表头居中

方法来自：https://bbs.csdn.net/topics/392215863

```html
<center>text</center>
```

|<center>数据类型</center>|<center>转换为true值</center>|<center>转换为false的值</center>|
|----|----|----|
|Undefined|-|undefined|
|Null|-|null|
|Boolean|true|false|
|String|任何非空字符串|""（空字符串）|
|Number|任何非零数字值（包括无穷大）|0和NaN|
|Object|任何对象|-|
